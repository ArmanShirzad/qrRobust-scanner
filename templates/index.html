How It Works

QR Robust Scanner differentiates itself from generic online tools by using a multi-stage decoding pipeline. Unlike client-side only scanners, we leverage server-side processing power to handle difficult images.

1. Upload

Image is securely uploaded via SSL.

2. Process

Image is normalized (grayscale, robust cropping) using PIL.

3. Decode

Parallel decoding using zxing-cpp and pyzbar libraries.

This project is open source. Check out the backend code on GitHub.
